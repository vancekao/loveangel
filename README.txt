愛天使正式網站前後台版

這份檔案包含：
- 前台網站首頁與多頁內容
- 後台管理頁 admin.html
- 共用網站資料檔與樣式
- 以你提供的主視覺圖片作為首頁主視覺背景

網站頁面：
- index.html 首頁
- about.html 關於愛天使
- special-education.html 特教認識專區
- news.html 最新消息
- event.html 園遊會專區
- sponsors.html 贊助合作
- contact.html 聯絡我們
- admin.html 後台管理

使用方式：
1. 將整個資料夾上傳到主機或靜態網站空間。
2. 直接開啟 index.html 即可瀏覽前台。
3. 開啟 admin.html 進入後台。
4. 預設密碼：admin1234

後台功能：
- 編輯首頁主視覺
- 編輯首頁特色區塊
- 編輯關於愛天使
- 新增/刪除/修改特教領域
- 新增/刪除/修改最新消息
- 新增/刪除/修改活動時程與攤位
- 新增/刪除/修改贊助單位
- 修改聯絡資訊
- 匯入/匯出 JSON
- 修改後台密碼

重要說明：
這版已經是完整的前台＋後台網站交付版，可直接展示、提案、內部管理與靜態部署。
目前資料是存在瀏覽器 localStorage，因此：
- 同一台電腦同一個瀏覽器會保留編輯結果
- 不同電腦不會自動同步
- 若要真正多人管理、雲端同步、圖片上傳、表單收件，需要再升級成資料庫版本


本版額外修正：
- 左上角 Logo 改為透明背景，並依電腦/平板/手機自動調整大小
- 修正 Logo 上傳後前台不顯示的問題
- 修正首頁報名資訊（含「我想參加活動」）圖片上傳後無法正常儲存的問題
